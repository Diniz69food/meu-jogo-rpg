# Motor de Jogo RPG - Turn-Based Engine

Este projeto é um motor simplificado de RPG baseado em turnos, desenvolvido para a disciplina de **PDS (Programação e Desenvolvimento de Software)** na **UFMG**.

## 🚀 Funcionalidades
- **Sistema de Classes:** Modelagem de personagens usando Orientação a Objetos.
- **Combate por Turnos:** Lógica de ataque e contra-ataque.
- **Gestão de Recursos:** Uso de itens (poções) com impacto imediato nos atributos.
- **Condição de Vitória:** Verificação em tempo real da vida dos personagens.

## 🛠️ Tecnologias
- Linguagem: **C++**
- Conceitos: Classes, Métodos, Passagem por Referência e Estruturas de Repetição.
